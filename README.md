# 💅 Sandhas Beauty Bar

A modern, responsive e-commerce and appointment booking web application built for **Sandhas Beauty Bar**. This platform allows clients to explore premium luxury products, purchase items, and seamlessly schedule beauty appointments online.

---

## ✨ Core Features

* **📅 Online Appointment Booking:** An intuitive booking system where clients can select a service (Nails/Hair), choose a date and time slot, and book an appointment instantly.
* **🛍️ Luxury Product Catalog:** A clean retail storefront displaying:
  * **Perfumes:** Premium, long-lasting fragrances.
  * **Hair:** High-quality bundles, wigs, and hair care products.
* **📱 Responsive UI/UX:** A fully optimized mobile-first design, ensuring clients get a luxurious experience whether browsing on a phone or desktop.
* **🔒 Admin Dashboard:** A secure backend interface for the owner to manage incoming bookings, update product inventory, and track client schedules.

---

## 🛠️ Tech Stack

| Layer | Technology | Purpose |
| :--- | :--- | :--- |
| **Frontend** | HTML5, CSS3, JavaScript | Semantic structure, sleek modern layouts, and dynamic UI elements |
| **Backend** | PHP | Secure server-side request processing, routing, and form validation |
| **Database** | MySQL | Relational data management for products, users, and appointment slots |
| **Server Environment** | XAMPP / Apache | Local development server infrastructure |
